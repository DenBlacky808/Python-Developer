# Background Repos Index

Этот индекс фиксирует старые учебные репозитории как контекст восстановления знаний. Они полезны как архив, фундамент и источник тем для повторения, но не являются источником для прямого копирования в production.

## Common Role Of Old Repos

- Архив: показывают, какие темы уже встречались в обучении.
- Фундамент: помогают вспомнить базовые понятия и ход мысли.
- Устаревший код: могут содержать старые версии, паттерны и настройки.
- Источник тем для повторения: подсказывают, какие блоки стоит освежить.
- Не production-шаблон: перед переносом любой идеи нужны modern review, обновление зависимостей, проверка безопасности и привязка к текущему проекту.

## How To Use This Index

- Сначала выбрать живой проект и конкретную задачу.
- Затем найти в старом фоне тему, которая помогает понять эту задачу.
- Повторить только нужный минимум, не весь курс целиком.
- Сделать один маленький артефакт: заметку, checklist, схему, вопрос или мини-PR.
- Не мержить старые PR и не переписывать старый код ради самого переписывания.
- Если тема пока не складывается, пометить ее как `unknown-later-decodable` и вернуться после практического примера.
